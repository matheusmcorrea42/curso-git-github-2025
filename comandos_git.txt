git init . - inicializa um repositório git na pasta atual
git status - mostra status do diretório
git add {arquivo}- o que eu quero que vá para o commit
git commit -m {mensagem} - faz o commit do arquivo do passo anterior 
git reset {arquivo} - tira o arquivo do staged, não commita
git checkout -b {nome da branch} - cria uma nova branch
git  branch - mostra a branch atual
git checkout {nome da branch} - seleciona a branch
git merge {nome da branch} - mescla as branchs
git branch -D {nome da branch} - deleta uma brach
